# Grilo Repair Shop - Nuxt + Firebase template (v2)

Upgraded version with BootstrapVue and full Admin OS CRUD.

Quick start:
1. Copy `.env.example` to `.env` and fill Firebase keys.
2. `npm install`
3. `npm run dev` (development)
4. `npm run generate` + `firebase init` + `firebase deploy` for hosting

Notes:
- Admin can create orders (OS) with services and upload images.
- Uses BootstrapVue for ready UI components.
